To use the Google Drive integration, you need to provide your own Google Cloud credentials. These are saved locally in your browser and are never sent to our servers.

How to get credentials:
  1. Go to Google Cloud Console
  2. Create a project and enable "Google Drive API"
  3. Create OAuth 2.0 Client ID (Web Application)
  4. Create API Key
  5. Add http://localhost:8000 and your GitHub Pages URL to "Authorized JavaScript origins"
